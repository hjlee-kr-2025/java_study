JAVA - 백엔드개발 - 자바 기초

- 프로그램구조가 class 로 이루어져 있습니다.
	화일이름과 class이름을 똑같이 하도록 만들어졌습니다.
	클래스이름(화일이름)은 첫글자가 대문자입니다.

- 변수, 변수의 자료형 (8가지)
	처음 선언할때부터 자료형에따라 선언하는 예약어가 다르다.
	변수가 처음부터 자료형이 정해져있고 자료형변경이 불가능합니다.
	1. 정수형 - 4가지
		- byte		: 1byte 메모리공간차지
		- short		: 2byte
		- int (기본)	: 4byte
		- long		: 8byte
	2. 문자형 - 1가지
		- char		: 2byte
	3. 실수형 - 2가지
		- float			: 4byte
		- double (기본)	: 8byte
	4. 논리형 - 1가지
		- boolean		: 1byte : true or false
		
** 이클립스의 console 에 출력하는 함수
	System.out.println(출력내용);
	: 출력내용 - 변수, 문자열("": 큰따옴표로 감싸야합니다)
	: 한글자를 감쌀때는 ''(작은따옴표) 를 사용합니다.
	: 출력내용에 변수를 사용할때는 변수의 자료형에 맞춰서 화면에 출력됩니다.
	
** 자료형의 형변환
- 자동형변환
	작은방에 큰방으로 값을 복사할때 자동으로 이루어집니다.
	* byte -> short -> int -> long
	* float -> double
	* int -> float -> double
	* long -> double
	* char -> int (코드값복사)
- 강제형변환
	큰방에서 작은방 값을 복사할때 자동으로 형변환이 안되어서
	개발자가 강제로 변환시키는 방식입니다.
	변수앞에 가로를 사용해서 안에 변경되는 자료형을 명시합니다.
	예)
	int iNum = 10;
	byte bNum = (byte)iNum; --> 강제 형 변환
	
** java프로그램이 처음시작되는 함수
public static void main(String[] args) {
	// 이곳에서 프로그램이 처음 시작됩니다.
}	

** 프로그램을 만들때는 프로그램이 실행되는 흐름을 이해하는 것이 중요합니다.
